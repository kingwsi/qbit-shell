# qbittorrent-shell
qbittorrent下载脚本，结合iOS快捷指令实现快速添加下载任务

## 使用
- 1.将脚本下载放在服务器/opt目录下
- 2.修改脚本里的qbittorrent账号密码，地址端口
- 3.下载快捷指令并修改，快捷指令需要自行修改服务器地址账号密码

`还有一种方案，对qbittorrent设置白名单或免密码，这样不用配置服务器脚本，直接可用快捷指令请求qbittorrent下载接口，考虑到安全性 这里不采用了`
